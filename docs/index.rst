**Oceanliner Documentation**
======================================

Introduction
------------

This package is being developed as part of a NASA-funded SWOT Science Team project to develop tools for planning in situ oceanographic campaigns that will take place after SWOT's launch, notably as part of the Adopt-a-Crossover program.
